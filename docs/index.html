Build a Tiny GPT From Scratch with NumPy

How it was built

Part 1: Tokenizer and Batching

Build a character-level vocabulary, encode/decode text, and sample training batches of (input, target) sequences.

Part 2: Parameter Initialization

Initialize all model parameters (embeddings, linear layers, layer norms, attention, MLP) using small random/Xavier-style init.

Part 3: Forward Primitives

Implement reusable forward ops: softmax, linear, ReLU, layer norm, causal mask, embedding lookup.

Part 4: Model Forward Pass

Compose primitives into self-attention, MLP, transformer blocks, and a full GPT forward pass producing logits.

Part 5: Loss and Backward Pass

Cross-entropy loss and manual backprop through every layer of the model, accumulating parameter gradients.

Part 6: Training Loop and Generation

Adam optimizer, training step, and autoregressive text generation with temperature sampling.
